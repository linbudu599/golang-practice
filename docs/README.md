## 语法

- 顶部包声明, 必须包含`main`
- 引入包, 如`fmt`实现了格式化IO的工具函数
- PrintIn自动增加换行字符`\n`
- 以大写字符开头的标识符即相当于被导出
- `{` 不能单独放在一行
- 字符串连接同样基于`"+"`
